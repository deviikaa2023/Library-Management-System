📚 Library Management System

## Overview
This is a Library Management System developed using Python and PostgreSQL. The project demonstrates Object-Oriented Programming (OOP) concepts along with SQLAlchemy ORM for database management.

## Features
- Add Books
- View Books
- Add Users
- Borrow Books
- Return Books
- Fine Calculation
- Book Availability Tracking
- SQLAlchemy ORM Integration
- PostgreSQL Database

## Technologies Used
- Python
- PostgreSQL
- SQLAlchemy
- Object-Oriented Programming (OOP)

## Concepts Used
- Classes and Objects
- Inheritance
- Polymorphism
- SQLAlchemy ORM
- Database Relationships
- CRUD Operations

## How to Run
1. Install Python.
2. Install PostgreSQL.
3. Install SQLAlchemy and psycopg2.
4. Configure the database connection.
5. Run the project using:

```bash
python library_sqlalchemy.py
```
